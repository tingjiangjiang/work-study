二进制：为什么计算机不能使用十进制或其他进制？因为计算机是一个电器设备，可以使用晶体管来控制有电或没电，就两个状态。二进制正好也是两个状态，正好和电的状态对应；通过多个晶体管的状态组合，产生一组数据：
	如：
		开关开关关开关关开
		1 0 1 0 0 1 0 0 1

	二进制的进位：
		0	1	2	3	4	5
		0	1	10	11	100	101
						2^2	2^2 + 0 + 2^0
八进制：
		2^3 = 8
		1	0	0	0
		2^3		
		需要3个2进制位
		
十六进制：
		2^4
		需要4个2进制位
十进制：
		二进制和十进制之间没有直接的换算关系
